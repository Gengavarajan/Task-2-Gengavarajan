# 📊 Student Grade Calculator

A Java-based Student Grade Calculator developed as part of my Java Programming Internship. This application calculates total marks, percentage, and grades based on the marks entered by the user, providing a quick and efficient way to evaluate academic performance.

## 🚀 Features

- Accepts marks for multiple subjects
- Calculates total marks obtained
- Computes average percentage
- Assigns grades automatically
- Displays result summary
- User-friendly console interface

## 🛠️ Technologies Used

- Java
- Object-Oriented Programming (OOP)
- Scanner Class
- Conditional Statements
- Arithmetic Operations

## ▶️ How to Run

1. Clone the repository:

   ```bash
   git clone <https://github.com/Gengavarajan/Task-2-Gengavarajan.git>
   ```

2. Open the project in IntelliJ IDEA, Eclipse, or any Java IDE.

3. Compile and run the Java file:

   ```bash
   javac StudentGradeCalculator.java
   java StudentGradeCalculator
   ```

4. Enter the marks when prompted to view the calculated results.
